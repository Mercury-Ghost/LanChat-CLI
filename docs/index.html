项目概述

LanChat CLI 是一个基于 TCP/TLS 通信协议的局域网聊天室命令行程序，采用客户端-服务器中心化架构设计。 项目提供全屏终端用户界面，支持群聊、私聊、房间管理、文件传输、在线用户列表、用户昵称修改等完整功能。

系统通过 TLS 加密通信、Argon2id 密码哈希与 JWT 令牌鉴权保障通信安全， 具备心跳检测、超时断开和断线重连能力，使用 SQLite 实现消息历史与用户数据的持久化存储。 架构上预留了去中心化（P2P）扩展接口。

核心功能

安全通信

TLS 1.2/1.3 加密、自签名 X.509 证书、Argon2id 密码哈希、JWT 令牌认证

聊天功能

房间群聊、点对点私聊、实时在线用户列表、消息历史记录存储与查询

文件传输

服务器中转模式、断点续传、最大 500MB 文件、传输进度实时显示

终端界面

基于 blessed 框架的全屏 TUI、多区域布局、彩色消息渲染、快捷键支持

连接可靠性

心跳检测（15秒间隔）、超时自动断开（10秒）、断线自动重连

数据持久化

SQLite 数据库、WAL 模式、历史消息自动推送（默认50条）

快速开始

安装依赖

# 克隆项目
git clone <repository-url>
cd LanChat-CLI

# 安装依赖
npm install

生成 TLS 证书

# 创建证书目录
mkdir -p certs

# 生成自签名证书（有效期10年）
openssl req -x509 -nodes -days 3650 -newkey rsa:2048 \
  -keyout certs/server.key \
  -out certs/server.crt \
  -subj "/CN=LanChat"

配置与编译

# 复制配置文件并编辑
cp .env.example .env

# 编辑 .env，设置 JWT_SECRET（至少32字符）

# 编译项目
npm run build

启动服务

# 终端1 - 启动服务端
npm run start:server

# 终端2 - 启动客户端
npm run start:client

客户端命令

命令 说明
/join #<room> 加入或创建指定房间
/leave 离开当前房间
/rooms 列出所有房间及在线人数
/msg <nick> <message> 向指定用户发送私聊消息
/nick <newNick> 修改当前用户昵称
/list 刷新当前房间在线用户列表
/history [count] 拉取历史消息（默认50条，最大200条）
/sendfile <nick> <path> 向指定用户发送文件
/quit 优雅断开连接并退出程序
