功能特性

为什么选择 LanChat CLI?

现代化的终端聊天体验，安全、高效、功能丰富

🔐

安全通信

TLS 1.2/1.3 加密，自签名 X.509 证书，Argon2id 密码哈希，JWT 令牌认证，全面保障通信安全。

💬

聊天功能

房间群聊、点对点私聊、实时在线用户列表、消息历史记录存储与查询，支持用户昵称修改。

📁

文件传输

服务器中转模式，支持断点续传，最大 500MB 文件，传输进度实时显示，文件元信息持久化。

🖥️

终端界面

基于 blessed 框架的全屏 TUI，多区域布局，彩色消息渲染，支持键盘快捷键与鼠标操作。

⚡

连接稳定

心跳检测机制（15秒间隔），超时自动断开（10秒），断线自动重连（指数退避策略）。

💾

数据持久化

SQLite 数据库存储，WAL 模式提升并发性能，历史消息自动推送，默认 50 条记录。

快速开始

5 分钟上手

简单几步，即可在局域网内开启聊天

1

克隆项目并安装依赖

# 克隆仓库
git clone <repository-url>
cd LanChat-CLI

# 安装依赖
npm install
2

生成 TLS 证书

# 创建证书目录
mkdir -p certs

# 生成自签名证书（有效期 10 年）
openssl req -x509 -nodes -days 3650 -newkey rsa:2048 \
  -keyout certs/server.key \
  -out certs/server.crt \
  -subj "/CN=LanChat"
3

配置环境并编译

# 复制配置文件
cp .env.example .env

# 编辑 .env，设置 JWT_SECRET 等

# 编译项目
npm run build
4

启动服务

# 终端 1 - 启动服务端
npm run start:server

# 终端 2 - 启动客户端
npm run start:client
命令参考

客户端命令

常用命令一览，快速掌握操作技巧

命令 说明
/join #<room> 加入或创建指定房间
/leave 离开当前房间
/rooms 列出所有房间及在线人数
/msg <nick> <message> 向指定用户发送私聊消息
/nick <newNick> 修改当前用户昵称
/list 刷新当前房间在线用户列表
/history [count] 拉取历史消息（默认 50 条，最大 200 条）
/sendfile <nick> <path> 向指定用户发送文件
/quit 优雅断开连接并退出程序

│   ├── services/          # 业务逻辑层
│   ├── TlsServer.ts       # TLS 服务器

│   ├── Database.ts        # 数据库管理
│   ├── UserManager.ts      # 用户管理
│   ├── RoomManager.ts      # 房间管理
