# 局域网命令行聊天室 - 使用指南

## 🚀 快速开始

### 1. 安装依赖

```bash
cd d:\LanChat-CLI
npm install
```

### 2. 构建项目

```bash
npm run build
```

### 3. 启动服务端

打开**第一个终端**运行服务端：

```bash
npm run start:server
```

启动成功后会显示：
```
服务器已启动，监听端口 9527...
```

### 4. 启动客户端

打开**第二个终端**运行客户端：

```bash
npm run start:client
```

## 📱 基础操作流程

### 第一步：注册账号

首次使用需要先注册账号：

```
=== 局域网聊天室 ===

登录(1) 还是 注册(2)? 2
昵称: Alice
密码: MyPass123

注册成功！请登录。
```

### 第二步：登录账号

注册成功后进行登录：

```

登录(1) 还是 注册(2)? 1
昵称: Alice
密码: MyPass123

登录成功！
```

### 第三步：发送消息

登录成功后，直接输入消息按回车即可发送：

```
=== 局域网聊天室 - #general ===
在线用户: Alice

[14:30:00] <Alice> 大家好！欢迎来到聊天室

========================================
> 在这里输入消息...
```

## 🔒 密码规则

密码必须满足以下要求：
- 长度在 **8-20** 个字符之间
- 必须包含至少一个**大写字母**（如 A-Z）
- 必须包含至少一个**小写字母**（如 a-z）
- 必须包含至少一个**数字**（如 0-9）

✅ 正确示例：`MyPass123`、`HelloWorld2024`、`Aa123456`
❌ 错误示例：`123456`（缺少字母）、`password`（缺少大写和数字）、`Pass1`（长度不足）

## 📋 命令列表

| 命令 | 说明 | 示例 |
|------|------|------|
| `/rooms` | 列出所有房间 | `/rooms` |
| `/join #房间名` | 加入指定房间 | `/join #tech` |
| `/leave` | 离开当前房间，返回 #general | `/leave` |
| `/msg <昵称> <消息>` | 发送私聊消息 | `/msg Bob 你好！` |
| `/nick <新昵称>` | 修改昵称 | `/nick Alice2` |
| `/list` | 查看当前房间在线用户 | `/list` |
| `/history [数量]` | 查看历史消息，默认50条 | `/history 20` |
| `/history @昵称 [数量]` | 查看与指定用户的私聊历史 | `/history @Bob 30` |
| `/passwd` | 修改密码 | `/passwd` |
| `/quit` | 退出聊天室 | `/quit` |
| `/help` | 显示帮助信息 | `/help` |

## 🌐 多用户使用方法

### 在局域网内多台电脑使用

1. **服务端电脑**：启动服务端
2. **客户端电脑**：
   - 确保与服务端在**同一局域网**内
   - 找到服务端电脑的 IP 地址（如：192.168.1.100）
   - 修改客户端的 `.env` 文件：
     ```env
     SERVER_HOST=192.168.1.100
     SERVER_PORT=9527
     ```
   - 启动客户端即可连接

### 在同一台电脑测试多用户

打开**多个终端**窗口，每个窗口启动一个客户端：

```bash
# 终端1 - 用户1
npm run start:client

# 终端2 - 用户2  
npm run start:client
```

## 🖥️ 界面说明

聊天室界面分为三个区域：

```
=== 局域网聊天室 - #general ===  ← 标题栏（显示当前房间名）
在线用户: Alice                    ← 状态栏（显示当前用户名）

[14:30:00] <Bob> 大家好！           ← 消息区域（按时间顺序排列）
[14:30:05] <Alice> 你好呀！
[14:30:10] [系统] Charlie 加入了 #general

======================================== ← 分隔线
> 输入消息...                       ← 输入区域（底部固定）
```

### 消息颜色说明

- 🟢 **绿色**：系统消息（如有人加入/离开）
- 🟡 **黄色**：私聊消息
- 🔵 **青色**：自己发送的消息
- ⚪ **白色**：其他用户发送的消息

## ⚠️ 常见问题

### Q1: 登录失败怎么办？

登录失败后会自动返回登录界面，提示错误原因：
- **密码错误**：请检查密码是否正确
- **昵称已在线**：该昵称已在其他地方登录
- **Token 已过期**：请重新登录

### Q2: 忘记密码怎么办？

目前版本暂不支持密码找回功能，请记住您的密码。

### Q3: 连接失败怎么办？

1. 检查服务端是否已启动
2. 检查客户端与服务端是否在同一局域网
3. 检查防火墙是否允许 9527 端口通信

### Q4: 如何退出聊天室？

输入命令 `/quit` 或按 `Ctrl+C` 退出。

## 🛠️ 配置说明

### 修改端口

修改 `.env` 文件：
```env
PORT=9527
```

### 启用 TLS 加密

```env
USE_TLS=true
TLS_CERT_PATH=certs/server.crt
TLS_KEY_PATH=certs/server.key
```

### 修改日志级别

```env
LOG_LEVEL=info  # 可选: debug, info, warn, error
```
